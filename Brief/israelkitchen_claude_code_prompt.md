# IsraelKitchen — Full Project Brief for Claude Code

## זהות המוצר
- שם: IsraelKitchen
- דומיין: israelkitchen.app
- תיאור: רשת חברתית למתכונים של ישראלים שגרים בחו"ל
- קונספט: משתמש מזין מתכון, המערכת מסדרת ומתרגמת מצרכים לשמות מקומיים, מזהה מצרכים נדירים ואיפה לקנות אותם, ומאפשרת שיתוף עם קהילה עולמית

## טכנולוגיה
- Frontend: React
- Backend/DB: Supabase — פרויקט: israelkitchen, Northeast Asia (Tokyo)
- Hosting: Vercel
- Claude Code: ~/claudecode/israelkitchen
- מפה: Leaflet.js + CartoDB Dark Matter tiles
- מודרציה AI: OpenAI Moderation API + Perspective API + AWS Rekognition
- Auth: Supabase Auth — Email + Google + Apple SSO

## מסכים שצריך לבנות

### 4.1 Feed קהילתי (עמוד ראשי)
- כותרת: IsraelKitchen + לוגו
- שורת חיפוש: "חפש מתכון, מצרך, מדינה..."
- פילטרים: הכל / בשרי / טבעוני / ארוחת בוקר / קינוחים / לפי מדינה עם דגלים
- כרטיסי מתכון: תמונה גדולה + overlay כהה + שם מתכון + שם משתמש + דגלים + עיר + סטטיסטיקות
- כפתור + צף לתחתית להוספת מתכון

### 4.2 עמוד מתכון בודד
- תמונה hero גדולה עם כפתורי חזרה ושיתוף
- שם מתכון, מחבר עם דגלים ועיר
- סטטיסטיקות: מנות / זמן הכנה / רמה / תגובות
- מצרכים: שם עברי + שם מקומי באנגלית + כמות + תגית "נדיר"
- כפתורי +/- למנות — משנה כמויות בזמן אמת
- תיבת "איפה לקנות מצרכים נדירים" עם מיקומים ספציפיים
- שלבי הכנה ממוספרים
- כפתור "הוסף לרשימת קניות"
- קרדיט למקור: נעול אם הגיע מלינק, אופציונלי אם הגיע מטקסט

### 4.3 מצב בישול
- פס התקדמות עם אחוז השלמה
- שלבים: נוכחי בולט, שעברו מעומעמים עם "הושלם — לחץ לחזרה", הבאים ממתינים
- טיימר מובנה: הפעלה / השהייה / איפוס
- כפתור "סיימתי שלב זה" + כפתור חזרה
- מסך סיום: "בתיאבון! 🍳"

שלב 1 — הזנה:
- כותרת: "מתכון חדש" (גדולה)

- Placeholder: "תשפכו פה את המתכון. אפשר להקליד, להדביק, להעתיק מהוואטסאפ של אמא... הכל בסדר. אנחנו נדאג לסדר את זה"
- כפתור: "יאללה, אפשר להתקדם" עם אייקון להבה

שלב 2 — עיבוד AI:

- שלבי עיבוד עם dots מתקדמים: זיהוי מצרכים / תרגום מקומי / מיפוי נדירים / סידור שלבים / תגיות

שלב 3 — תגיות + שיתוף:

- כפתור: "שמרו את המתכון"

שלב 4 — שיתוף לחברים:

### 4.6 מפה עולמית
